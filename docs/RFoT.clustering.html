RFoT.clustering.gmm.GaussianMixture_get_labels(M_r, n, random_state)[source]¶

Extracts the cluster labels for KRUSKAL tensor M_r for n clusters. M_r is the rth component of the KRUSKAL tensor M_i. :param M_r: dict, KRUSKAL tensor's first latent factor. :param n: int, number of clusters.

RFoT.utilities.bin_columns module¶

RFoT.utilities.bin_columns.bin_columns(df: pandas.core.frame.DataFrame, bin_max_map, dont_bin: [], bin_scale=1.0)[source]¶

Maps the tensor dimensions to bins for indexing the tensor. The number of bins is specified by the number of unique entries for the given dimension in the dataset, and scaled if specified. :param df: pd.DataFrame, :param dont_bin: str, name of the feature to not create bins.

Used to avoid mapping feature values to bins for the feature used as the tensor entry.
